Solar Knowledge Project

Advanced Machine Learning for Solar Flare Prediction

Project Overview

The Solar Knowledge Project aims to improve solar flare prediction using advanced machine learning techniques. By leveraging time-series data from solar observations, we develop models that can accurately predict solar flares 24-72 hours in advance.

Key Features

Multi-Class Prediction

Predicts C, M, and X-class solar flares with improved accuracy compared to existing approaches.

Time-Series Forecasting

Uses historical solar data to forecast flare activity 24, 48, and 72 hours in advance.

Multimodal Learning

Combines multiple data sources including GOES X-ray flux and SDO imagery to improve predictive capability.

Our Models

The project includes several model variations:

  • SolarKnowledge Base: Our foundation model for solar flare prediction
  • Multimodal SolarKnowledge: Enhanced with image data processing capabilities
  • Ablation Studies: Multiple variations to test the importance of different model components

Results

Our models have demonstrated significant improvements over baseline approaches:

  • Improved precision and recall for M and X-class flare prediction
  • Better lead time for advance warnings
  • Reduced false positive rate while maintaining high detection capability
